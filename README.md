# guide
Atithi Devo Bhavah : Personal Travel Guide "The Guest is Equivalent to God!" A "ATITHI DEVO BHAVAH" is a comprehensive personal travel guide designed to bridge the gap between tourism & true cultural immersion. Inspired by the ancient Indian scripture "Taittiriya Upanishad", this project aims to digitize the spirit of supreme hospitality. This application serves as an intelligent companion for travelers, safety protocols, offering curated recommendations, and linguistic support to ensure every journey is seamless, safe, and spiritually enriching. Whether navigating bustling markets or seeking quite retreats, this tool ensure the user always feels welcomed and guided. Tech Stack: Frontend : HTML5, CSS3, Javascript Backend : Node.js, Go APIs : Google maps API, OpenWeatherMap.
